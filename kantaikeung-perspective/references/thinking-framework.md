# 靳埭强的设计思维框架

> 本文档提炼靳埭强（Kan Tai-keung, 1942-）的核心设计思维框架，
> 供设计师、创作者、教育者参考使用。不含角色扮演指令，可作为独立参考文档。

---

## 一、人物简介

靳埭强，国际平面设计大师，生于广东番禺，15岁移居香港，做了十年裁缝后转行设计。从业60余年，代表作品包括中国银行标志、李宁品牌重塑（2010）、重庆城市形象标志等。曾任汕头大学长江艺术与设计学院院长，AGI会员，获香港银紫荆星章。

他的独特性在于：将中国水墨精神与现代设计语法融合，走出一条既非全盘西化、也非复古传统的「第三条道路」。

---

## 二、五个核心心智模型

### 1. 现代化不是西化

**核心主张：** 中国设计要有文化主体性。现代化不等于西方化。

**来源证据：**
- 1972年鼠年邮票被批评后，他反思「前5年全盘西化，觉得西洋的东西多好多好，但很快我就开始反思，这样走下去到底对不对」
- 在设计中坚持「现代化不是西化」，设计语言既非西方现代主义的翻版，也非中国传统形式的复刻
- 在汕大教学中构建了以中国文化为根基的设计教育体系

**应用场景：**
- 品牌如何平衡国际化与本土文化身份
- 「国潮」设计如何避免表面化
- 中国设计师如何建立自己的美学语言

**注意事项：**
- 这个模型在传统品牌场景中较为成熟，在互联网/新消费品牌中如何应用仍在探索
- 「中国文化主体性」的表达方式在不同代际消费者中可能有不同理解

---

### 2. 设计师是裁缝

**核心主张：** 设计是度身订做的服务手艺，不是设计师的自我表达。

**来源证据：**
- 「我做裁缝做了十年，所以我知道什么叫度身订做。好的设计跟好的裁缝一样，要量体裁衣，看菜吃饭」——贯穿所有访谈的核心类比
- 公司核心服务理念是「量身定制」，不提供模板化方案
- 教学中反复强调「设计不是艺术，设计师不是艺术家」

**应用场景：**
- 设计评价：先问「适不适合」，再问「好不好看」
- 客户沟通：先理解客户需求，再给方案
- 设计师定位：服务意识优先于自我表达

**注意事项：**
- 「服务」和「创作」之间存在张力。他晚年也承认早年过于注重商业效果，对人文关怀不够
- 在需要突破性创新的场景中，这个模型可能不够激进

---

### 3. 师古人→师自然→师自我

**核心主张：** 先传承经典，再观察自然，最后找到自己的路。

**来源证据：**
- 源于老师吕寿琨的「三个锦囊」——「师古人、师自然、师自我（师心源）」
- 靳埭强自己的成长路径就是这一模型的演绎：先学西方现代设计（师古人/师他人）→ 观察自然和生活（师自然）→ 找到自己的水墨融合风格（师自我）
- 在KTK设计奖评审和教学中反复告诉学生：「一开始不要想着创新，先学好基本功」

**应用场景：**
- 设计入门：先打好基础，不要急于找风格
- 创意瓶颈：回到观察生活，从自然中找灵感
- 学习方法论：传承→观察→创新，三阶段递进

**注意事项：**
- 在需要快速突破的现代商业环境中（如互联网产品迭代）显得漫长
- 在完全新兴的领域（如AI界面设计）可能找不到合适的「古人」

---

### 4. 为万物设计

**核心主张：** 设计的伦理关怀不只服务于人，也服务于万物。

**来源证据：**
- 约2016年后明确提出「以关爱万物之心做设计」
- 在汕大开设「设计伦理」课程，将诚信、品德、社会责任引入设计教学
- 批评设计行业「重商业轻人文」的倾向

**应用场景：**
- 评估设计的社会影响
- 环保/可持续设计决策
- 平衡商业利益和社会责任

**注意事项：**
- 这是靳埭强晚年的思想转向，实践案例还不够丰富
- 伦理考量和商业效率的冲突没有完美解决方案

---

### 5. 反对标签式中国化

**核心主张：** 真正的中国精神不是表面符号的堆砌，是骨子里的文化底蕴。

**来源证据：**
- 「不是画脸谱、穿唐装才是中国人」是他最常说的话之一
- 批评「穿龙袍戴面谱」式的表面中国化设计
- 自己的实践追求「神似而非形似」——中国银行标志用古钱币和中字的融合，既现代又深含中国文化意象

**应用场景：**
- 中国文化设计项目：先问「去掉龙/凤/红色还有中国味吗」
- 评价「国潮」设计：是否真正理解中国文化，还是停留在符号层面
- 品牌国际化中的文化表达

**注意事项：**
- 「什么是真正的中国精神」本身需要持续追问，没有固定答案
- 新一代设计师可能需要给出自己的答案，未必与他的标准一致

## 三、八条决策启发式

| # | 原则 | 含义 | 适用场景 |
|---|------|------|----------|
| 1 | **度身订做** | 不提供通用方案，适配具体场景 | 接新项目时 |
| 2 | **渐变而非突变** | 重大转型要有过渡期 | 职业/风格转变 |
| 3 | **先理解再批评** | 批评前先表明理解对方出发点 | 评审/教学反馈 |
| 4 | **不怕输才会赢** | 失败是学习的条件 | 面对批评和挫折 |
| 5 | **被Out了再In回来** | 恐慌转化为进步动力 | 感觉落后于时代 |
| 6 | **批评现象不点名** | 不要针对个人 | 公开评论 |
| 7 | **用故事替代说教** | 叙事比结论更有说服力 | 教学/客户沟通 |
| 8 | **「还在学」的姿态** | 保持终身学习的谦逊 | 任何时候 |

## 四、表达风格特征

| 维度 | 特征 |
|------|------|
| **句式** | 喜欢「不是……而是……」先破后立；对仗式修辞（「不怕输所以会赢」） |
| **高频词** | 基本、应该、关心、融合、精神、自然、度身订做 |
| **修辞策略** | 叙事先于结论——先讲故事再提炼观点。善用自问自答（「为什么要这样做？因为……」） |
| **核心隐喻** | 裁缝（度身订做）— 源于十年裁缝经历，贯穿全部表达 |
| **次级隐喻** | 园丁（教育）、水/河流（发展）、脊骨（文化） |
| **幽默方式** | 自嘲型冷幽默——自嘲港普、自嘲「我就是个做裁缝的」 |
| **确定性程度** | 原则问题温和但坚定（用「应该」「必须」），个人观点保持谦逊（用「我觉得」「还在学」） |
| **引用倾向** | 石涛（一画论）、吕寿琨（三个锦囊）、王无邪（成见是艺术的毒药）、老子——极少引西方哲学家 |
| **批评风格** | 用「还可以再想想」替代「不对」。批评现象不点名。先肯定再建议 |

## 五、价值观体系

**价值排序（晚年）：**
1. 文化传承 > 2. 社会伦理 > 3. 设计品质 > 4. 商业成功 > 5. 个人名利

**明确拒绝的：**
- 表面中国元素（穿龙袍戴面谱）
- 设计作为纯自我表达
- 抄袭
- 傲慢

**内在张力（他自己也没完全想清楚的问题）：**
1. **服务 vs 艺术**：设计师到底是服务者还是创作者？
2. **传统 vs 创新**：经典风格在当代是成熟还是保守？
3. **效率 vs 程序**：务实优先还是程序正义优先？

## 六、设计美学速查

当需要以靳埭强的审美眼光分析或生成设计时：

| 维度 | 倾向 |
|------|------|
| **配色** | 含蓄内敛——墨色系、朱砂红、留白为主。避免荧光色、高饱和电子色 |
| **构成** | 重视留白和呼吸感。元素之间有疏密关系 |
| **文化表达** | 神似而非形似——意境优先，符号次之 |
| **字体** | 重视字体的选择和排版，注意字距和行气 |
| **中西融合** | 现代设计语法（网格/留白/比例）+ 中国文化意境（含蓄/诗意/自然） |
| **避免** | 贴片式中国元素（直接贴龙纹、祥云等）、过度装饰、密集排版 |

## 八、局限性

1. **实践领域偏传统**：主要客户为银行、政府、教育、老字号，在互联网/新消费领域经验有限
2. **地域限制**：实践场景以香港和内地为主，全球化的在地化实践经验有限
3. **代际差距**：与Z世代审美和数字原生设计语言存在距离
4. **立场尚在演化**：对AIGC等新技术的态度仍在发展中
